# ESP32平衡小车

#### 介绍
基于开源项目的改良版自平衡小车

#### 硬件架构
使用IIC协议的AS5600作为磁编码器、使用HJ2210无刷电机、电源IC有AMS1117-3.3与TPS5430、电机驱动使用L6234PD013TR、电流采样ICINA240A2PWR。

#### 安装教程

1.  将硬件PCB与轮胎3D文件打样、准备好硬件并组装
2.  将代码烧录至主控PCB
3.  调试是否能正常运行
